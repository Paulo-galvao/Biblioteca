# Biblioteca
Sistema web do zero ao deploy com javascript, node e postgresql.

# Sobre o Projeto

API que simula uma biblioteca virtual. O seu principal objetivo é a prática de conceitos do desenvolvimento web, como autorização, autenticação, validação de dados e manipulação de banco de dados relacional, atráves de um projeto simples. 

Utilizei o serviço render para hospedar a API e o serviço Neon para usar um banco de dados em nuvem.
Assim fica tudo pronto para ser consumida pelo frontend que desenvolvi em React.

Url da Api https://biblioteca-1-hb53.onrender.com.

Url do FrontEnd https://readmorebooks.netlify.app/.

Repositório do FrontEnd https://github.com/Paulo-galvao/Biblioteca_FrontEnd.
